# StockRobo


## Setup

Activate the virtual environment

#### OSX
On MacOS machine
```sh
$ . venv/bin/activate
```

#### Windows
On windows machine
```sh
$ .\venv\Scripts/activate.bat
```


## Install dependencies
Install project dependencies using
```sh
$ pip install -r requirements.txt
```


## Start the script
```sh
$ make start
```
